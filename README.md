# Система управления отелем

Это веб-приложение для управления отелем, разработанное на Python с использованием фреймворка Django. Система предоставляет функционал для управления номерами, бронированиями, заездами/выездами и ценами.

## Функциональность

1. **Управление номерами**
   - Просмотр доступных типов номеров
   - Отслеживание количества свободных номеров в реальном времени

2. **Бронирования**
   - Просмотр списка всех бронирований
   - Отмена бронирований
   - Отслеживание статуса бронирований

3. **Заезды/Выезды**
   - Регистрация фактических заездов и выездов
   - Просмотр истории заездов/выездов

4. **Цены**
   - Просмотр цен по дням недели
   - Информация о скидках при длительном бронировании

## Требования

- Python 3.8 или выше
- Django 4.2.7 или выше

## Установка

### Вариант 1: Использование скрипта установки (рекомендуется)

1. Запустите файл `setup_project.bat` от имени администратора
2. Следуйте инструкциям на экране для создания виртуальной среды, установки зависимостей и создания базы данных

### Вариант 2: Ручная установка

1. Клонируйте репозиторий:
   ```
   git clone <url>
   ```

2. Перейдите в директорию проекта:
   ```
   cd hotel_management_system
   ```

3. Создайте виртуальное окружение:
   ```
   python -m venv venv
   ```

4. Активируйте виртуальное окружение:
   - В Windows:
     ```
     venv\Scripts\activate
     ```
   - В macOS/Linux:
     ```
     source venv/bin/activate
     ```

5. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```

## Настройка базы данных

1. Создайте миграции:
   ```
   python manage.py makemigrations
   ```

2. Примените миграции:
   ```
   python manage.py migrate
   ```

## Создание суперпользователя

Для доступа к панели администратора создайте суперпользователя:
```
python manage.py createsuperuser
```

## Запуск сервера разработки

### Вариант 1: Использование скрипта запуска (рекомендуется)

Запустите файл `run_project.bat`

### Вариант 2: Одна команда для всего (рекомендуется)

Запустите скрипт `start_project.py`, который автоматически:
- Создаст виртуальную среду
- Установит все зависимости
- Создаст и применит миграции базы данных

```
python start_project.py
```

### Вариант 3: Ручной запуск

```
python manage.py runserver
```

Сервер будет доступен по адресу: http://127.0.0.1:8000/

## Использование

Система имеет следующие разделы:

1. **Главная** - Обзор системы и ее функций
2. **Управление номерами** - Просмотр доступных номеров
3. **Бронирования** - Управление бронированиями
4. **Заезд/Выезд** - Регистрация заездов и выездов
5. **Цены** - Просмотр цен и скидок

## Важно

Для бронирования номера необходимо связаться с ресепшеном по телефону. Веб-интерфейс предназначен только для сотрудников отеля.
