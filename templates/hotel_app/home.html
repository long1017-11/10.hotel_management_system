Система управления отелем

Добро пожаловать в систему управления отелем

Эта информационная система предназначена только для использования на стойке регистрации отеля.

Функции системы:

  1. Для бронирования номера, пожалуйста, свяжитесь с ресепшеном по телефону.
  2. Типы номеров: стандартные номера, комфортные номера и люксы. Типы размещения: одноместные, двухместные и трехместные номера. Во все номера при необходимости можно добавить детскую кроватку.
  3. Цены на проживание различаются в зависимости от дня недели. Скидки доступны при длительном бронировании.
  4. Система позволяет отменить бронирование и записать фактические даты заезда и выезда.
  5. Управление номерами: можно в реальном времени видеть оставшиеся типы номеров.
Важно: Для бронирования номера необходимо связаться с ресепшеном по телефону.
